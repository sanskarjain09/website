Quick Start

Download the project and install dependencies once. Then run your .st files from anywhere using the DSL CLI. 

git clone https://github.com/sanskarjain09/finalyeardsl.git
cd finalyeardsl
npm install
npm run dev -- example/todo-app.hhtml 3000

Core Syntax

Adbhoya DSL supports app structure, state, backend API, and UI in the same file. 

<app>
  <state name="count" value="0" />
  <api route="/ping" method="GET">
    return { ok: true };
  </api>
  <client>
    <button click="count = Number(count) + 1">
      Count: {{count}}
    </button>
  </client>
</app>

Run Any .st File

After global linking, you can run your DSL app from any folder. 

tododsl run /absolute/path/to/your-app.st 3000

If you are using local command shortcut: runst /absolute/path/to/your-app.st 3000
